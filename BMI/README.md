### MVC 패턴이란?

Model - View - Controller로 파일들을 나눈 디자인 패턴이다.<br>
Model에서는 비즈니스 로직(화면과 전혀 관련이 없는), View에서는 UI(사용자 화면), Controller에서는 모델의 정보를 어떻게 뷰에 표시할지를 해석해서 전달하는 중재자 역할을 한다.
